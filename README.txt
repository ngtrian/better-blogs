Installation instructions
============================================================
1. Upload all the files inside the "upload" folder to your XenForo
2. Add the Mod in your AdminCP
	a) Go to your Admin Control Panel
	b) Click on "Add-ons"
	c) Click on "Install Add-on"
	d) Select the xml file and click on Install
	
Upgrade instructions
============================================================
1. Upload all the files inside the "upload" folder to your XenForo
2. Upgrade the Add-On in your AdinCP
	a) Go to your Admin Control Panel
	b) Click on "List Add-ons"
	c) Next to the Groups add-on, select Controls -> Upgrade
	d) Select the xml file from the package and click on Upgrade
